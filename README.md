Value
=====

Value is a support library for value objects in Java.

Download
--------

Add a dependency to Value:

    <dependency>
      <groupId>org.jvirtanen.value</groupId>
      <artifactId>value</artifactId>
      <version><!-- latest version --></version>
    </dependency>
